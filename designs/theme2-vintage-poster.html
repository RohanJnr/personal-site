Bike Tracker App

April 8, 2026 · projects/

I've been working on a real-time GPS tracker for my motorcycle rides. The goal is to log routes, speed, and elevation data using a Raspberry Pi Zero mounted on the bike.

Architecture

The system has three main components connected via a local message bus. See Home Server Setup for the backend details.

struct RideData {
    timestamp: u64,
    lat: f64,
    lon: f64,
    speed_kmh: f32,
    elevation_m: f32,
}
ⓘ Note

The GPS module needs a clear view of the sky. Mount it on the tail, not under the seat.

Next steps: integrate with the mapping API to render ride overlays on a web dashboard.
